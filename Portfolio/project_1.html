GekCon

2023г.

Задача

Создать оформление для интернет-портала, на котором пользователи смогут находить новости и события по гик-тематике и собираться в сообщества по интересам.

Цвета

Цветовая палитра сайта состоит из сочетания красного, желтого и черногоцвета. Такая палитра отсылает пользователей к ярким раскадровкам графичиских новелл

Шрифт

В качестве основного шрифта был выбран Inter. Несмотря на то, что вся гик-тематика носит неформальный характер, на сайте присутсвует большое количество текста. Данный шрифт сможет не нагружать пользователя

Элементы

На страницах сайта присутсвует большое количество похожих блоков. Чтобы не перенагружать их дизайн было принято решение разделять данные блоки сплошными линями

Общий стиль

Дизайн сайта старается балансировать между броскостью комиксов и сдержанностью новостных колонок. Для этого, броскость цвета на основных блоках сведена к минимуму.
